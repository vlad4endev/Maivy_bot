# Медиафайлы для бота

Положите сюда файлы для приветственного экрана:

| Файл | Назначение | Рекомендации |
|------|------------|--------------|
| `welcome.jpg` | Фото профиля бота + fallback, если нет видео | JPG/PNG, до 10 МБ, квадратное |
| `welcome-video.mp4` | Кружочек (Telegram) / видео (MAX) | MP4, квадратное 1:1, до 60 сек |

Если видео отсутствует, бот покажет фото или текстовое меню.

Для Telegram можно задать `TELEGRAM_VIDEO_NOTE_FILE_ID` в `.env` после первой загрузки — тогда файл не нужен при каждом запуске.
